# Mirobot PythonSDK-安装指南







## 安装构建好的库



Ubuntu:

```bash
sudo pip3 install mirobot-py
```

Windows

```bash
pip install mirobot-py
```



## 从源码安装 


克隆代码
```bash
git clone git@github.com:mushroom-x/mirobot-py.git
```

Windows下的安装
```bash
python .\setup.py install 
```

Ubuntu下的安装
```bash
python3 .\setup.py install 
```
